# 💎 Gem Puzzle (Bejeweled Clone)

TypeScript + Canvas로 만든 모바일/데스크탑 지원 보석 퍼즐 게임.

## 플레이 방법

보석을 인접한 칸과 교환해 3개 이상 같은 종류를 일렬로 맞추세요.
- **터치/마우스 탭**: 보석 선택 후 인접 보석 선택으로 교환
- **스와이프/드래그**: 원하는 방향으로 밀어 교환

---

## 게임 모드

### 🔷 퍼즐 모드
- 레벨 1~100, 단계별 난이도 증가
- 시간 제한 내에 목표 점수 달성 시 다음 레벨 진출
- 보석 매치 시 시간 보너스
- 레벨별 1~3성 평가 (IndexedDB 저장)
- 일시 정지 지원
- **레벨 순차 잠금**: 직전 레벨 클리어 후 다음 레벨 해금
- **이어하기**: 앱 재오픈 시 마지막 플레이 레벨에서 이어하기 가능

### 🧘 명상 모드
- 시간 제한 없는 자유 플레이
- 진행 상황 자동 저장 (IndexedDB)
- 언제든 이어서 플레이

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 반응형 UI | 모바일/폴더블/데스크탑 화면 자동 대응 |
| 이모지 보석 | 레벨에 따라 3~12종 이모지 보석 (💎⭐🔥💧🍀🌙⚡🌸✨🔮🌈👑) |
| 터치 지원 | 탭·스와이프·스크롤 모두 지원 |
| 사운드 | Web Audio API 기반 매치 효과음 |
| 데이터 저장 | IndexedDB (레벨 기록, 진행상황, 게임 저장) |
| 이어하기 | 재방문 시 이어하기/새로 시작 선택 화면 |
| 레벨 잠금 | 이전 레벨 클리어 후 다음 레벨 해금 |
| 모던 UI | 딥 스페이스 테마, 그라디언트, 글로우 효과 |

---

## 기술 스택

- **언어**: TypeScript → 바닐라 JavaScript (ES6)
- **렌더링**: HTML5 Canvas 2D API
- **저장소**: IndexedDB (puzzle_records, zen_save, game_save)
- **사운드**: Web Audio API
- **외부 라이브러리**: 없음

---

## 빌드 & 실행

```bash
# TypeScript 컴파일
tsc game.ts --target es6

# 로컬 실행 (index.html을 브라우저로 열기)
open index.html
```

## 개발 요구사항 (원본)

```
1. Bejeweled clone을 만들고 싶어
2. TypeScript로 구현을 해주세요
3. 바닐라 자바스크립트로만 구성이 되어야 해
4. Canvas를 이용해서 만들어줘
5. 퍼즐 모드, 명상 모드가 있었으면 좋겠어
6. 퍼즐 모드는 1-100단계까지 단계가 있었으면 좋겠어
7. 퍼즐 모드는 단계가 올라갈수록 난이도가 높아져
8. 퍼즐 모드는 시간 제한이 있고, 블록을 없앨 때마다 시간이 증가해
9. 퍼즐 모드는 일시 정지 모드가 있어
10. 퍼즐 모드는 각 판에서 얻은 점수에 따라서 별을 1-3개까지 보여줘
11. 퍼즐 모드의 모든 기록은 브라우저의 내장 DB에 저장되어야 해
12. 명상 모드는 시간 제한 없이 천천히 플레이하는 모드야
13. 진행상황은 브라우저의 내장 DB에 저장되고, 이어서 플레이가 가능해
14. 마우스와 터치스크린을 지원해야 해
15. 블록이 사라질 때마다 경쾌한 소리가 나야 해
```
