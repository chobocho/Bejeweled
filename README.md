# Bejewled Clone

## How to make
~~~
1. Bejewled clone을 만들고 싶어
2. Typescript로 구현을 해주세요
3. 바닐라 자바스크립트로만 구성이 되어야해
4. Canvas를 이용해서 만들어줘
5. 퍼즐모드, 명상모드가 있었으면 좋겠어.
6. 퍼즐 모드는 1-100단계까지 단계가 있었으면 좋겠어.
7. 퍼즐 모드는 단계가 올라 갈수록 난이도가 높아져.
8. 퍼즐 모든는 시간 제한이 있고, 블록을 멊앨 때마다 시간이 증가해.
9. 퍼즐 모드는 일시 정지 모드가 있어.
10. 퍼즐 모드는 각 판에서 얻은 점수에 따라서 별을 1-3개 까지 보여줘.
11. 퍼즐 모드의 모든 기록은 브라우저의 내장 DB에 저장 되어야 해.
12. 명상 모드는 시간 제한이 없이 천천히 플레이하는 모드야.
13. 진행상황은 브라우저의 내장 DB에 저장 되고, 이어서 플레이가 가능해
14. 마우스와 터치 스크린을 지원해야 해.
15. 블럭이 사라질때 마다, 경쾌한 소리가 나야해.
~~~

## Build & Run
tsc game.ts --target es6
